## Reduce number of requests

**Join multiple assets into a single one.**

* JS + JS: *Concat*
* IMG + IMG: *Sprites*
* CSS + CSS: *Concat*
* CSS + IMG/FONT: *DataURL*
* HTML + IMG: *DataURL*
* JS + CSS: *`doc.createElement("style")`*
* JS + HTML: *`inlineHTML`*
* HTML + JS: *`<script>`*
* HTML + CSS: *`<style>`*
